使い方

/ how_it_works.md
01
新規マップを開く
ホームから「新しいマインドマップ」を開く。
02
起点ワード → 自動生成
ツールバーに起点ワードを入れて「自動生成 (cascade)」を押すと、AIが関連語のネットワークを一括生成。
03
ダブルクリックで拡張
あるいは、ノードをダブルクリックするとその語の連想を1世代だけ追加 (expand)。
04
手動で編集
ノードはドラッグで配置、ダブルクリックで編集、パネルから子ノード追加や削除も可能。
05
共有URLで共同編集
「共有」ボタンのURLを他の人に渡すと、リアルタイムで一緒に編集できる。
06
PNG / SVG で出力
完成したらPNGまたはSVGでエクスポート。
// 02

機能

/ features.json
AUTO_GEN
自動生成
word-api (chiVe) から日本語の連想語を取得し、意味的ネットワークを即座に合成。
SYNC
リアルタイム共同編集
Yjs CRDT で競合なし同期。共有URLを渡すだけで複数人が同時編集できます。
EXPORT
画像エクスポート
PNG / SVGで出力。スライドや資料にそのまま貼り付け可能。
ANON
匿名でもOK
ログインなしで作成開始。後からアカウントを作れば保存・同期もできます。
CONFIG
カスタマイズ
類似度の閾値・品詞フィルタ・レイアウトアルゴリズム・配色テーマを自由に。
